Initialize Phases {#initializephase}
======

## Group (Subgroup) ##
Image Import Filters (Input)


## Description ##
Assigns an unknown phase to every voxel
----------


## Parameters ##


| Name             |  Type  |
|------------------|--------|
| Phase            | String |
| Minimum Value    | Int    |
| Maximum Value    | Int    |


## Required DataContainers ##

Voxel

## Required Arrays ##

None


## Created Arrays ##


| Type | Default Array Name |
|------|--------------------|
| Int  | Cell Phases        |
